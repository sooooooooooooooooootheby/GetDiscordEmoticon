# GetDiscordEmoticon

This tool can help you download emoticon from any server on Discord

click here [Get Discord Emoticon](https://dcemoticon.s22y.moe/)

# Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Compile and Minify for Production

```sh
pnpm build
```
